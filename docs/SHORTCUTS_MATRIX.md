# Shortcuts Matrix

## 목적
- 전역 단축키를 단일 기준으로 관리해 충돌/중복을 방지한다.
- 도움말 모달, 실제 핸들러, QA 체크리스트의 기준을 일치시킨다.

## 기준일
- 작성일: 2026-03-02
- 기준 파일: `src/utils/shortcuts.ts`, `src/App.tsx`, `src/components/ShortcutsModal.tsx`

## 충돌 방지 규칙
1. 전역 단축키는 `isEditableKeyboardTarget` 통과 시 비활성화한다.
2. 모달/시트가 열려 있으면 전역 단축키를 비활성화하고 해당 레이어 핸들러에 위임한다.
3. 토글형 단축키(`R`, `F`, `Ctrl/⌘+B`, `Ctrl/⌘+,`, `Ctrl/⌘+Shift+S`, `?`)는 반복 입력을 무시한다.
4. 단축키 도움말(`ShortcutsModal`)은 `src/utils/shortcuts.ts`를 단일 소스로 사용한다.

## QA 체크 포인트
- `Ctrl/⌘+K`로 검색 모달을 열었을 때 `F`, `R`, `?` 입력이 차트/화면에 영향 주지 않는지 확인
- `Esc`가 모달 닫기와 전역 패널 닫기를 컨텍스트에 맞게 분리 처리하는지 확인
- 데스크톱에서 `Ctrl/⌘+B`, `Ctrl/⌘+,`가 의도대로 무시되는지 확인
- `Ctrl/⌘+J`로 명령 허브를 열어 `Enter`로 첫 액션이 실행되는지 확인
