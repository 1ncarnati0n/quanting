# TradingView Feature Implementation Plan (Quanting)

문서: `docs/tradingview-feature-gap-analysis.md` 기준 구현 계획/진행 현황

## 목표

- Phase 1 기능을 우선 완성하여 실사용 체감 개선
- Phase 2~4는 아키텍처를 유지하며 순차 확장

## 진행 현황 (2026-02-28)

### Phase 1 — 핵심 분석 도구 강화

- [x] 추가 지표: `VWAP`, `ATR`, `Ichimoku`, `Supertrend`, `Parabolic SAR`
- [x] 기본 드로잉: 수평선 / 추세선 / 피보나치 되돌림 (+ 측정도구)
- [x] 가격 알림: 가격선 + 트리거/이력 + 브라우저 시스템 알림
- [x] 로그/% 스케일 + 자동/역전 스케일
- [x] 차트 비교(Overlay): 비교 심볼 오버레이 + 정규화 옵션

### Phase 2 — 프로 트레이딩 UX

- [x] 멀티 차트 레이아웃(1/2/4 분할 기본형)
- [x] 전략 성과 요약(승률/평균수익률 기본형)
- [x] 볼륨 프로파일 기본형(가격 구간 볼륨 오버레이)
- [x] 눈금자(Measure) 도구
- [x] 매수/매도 구간 배경 시각화

### Phase 3 — 고급 기능

- [x] 바 리플레이(순차 재생 + 속도/스크럽)
- [x] WebSocket 실시간 데이터(crypto, Binance)
- [x] 기술적 스크리너 기본형(워치리스트 규칙 스캔)
- [x] 기술적 스크리너 고도화(다중 조건/AND·OR/정렬/프리셋 저장)
- [x] 드로잉 도구 확장(고급 패턴: 피치포크/갠/엘리엇/하모닉)
  - 현황: 기본 확장(평행채널/직사각형/텍스트 주석) + 고급 패턴 렌더링/미리보기 완료
  - UX: 드로잉 선택 강조 + 선택 삭제(Delete 우선 삭제) + 개별/전체 삭제
- [x] 워크스페이스 저장/불러오기(Export/Import)

### Phase 4 — 데이터 확장

- [x] 외환(Forex)
  - 현황: `MarketType.forex` 추가, Yahoo Finance(`=X`) 심볼 조회/스냅샷/캐시 연동 완료
- [x] 금/은 + 한국 ACE ETF 목록 확장
- [ ] 선물/옵션 (요청 범위에서 보류)
- [x] 재무제표 오버레이(기본형)
- [x] 커스텀 인터벌 리샘플링

## 구현 원칙

- 기존 아키텍처 유지: `Zustand + Tauri IPC + Rust TA engine + lightweight-charts`
- 새 기능은 기본값 `OFF`로 도입 후 점진 활성화
- 렌더 부담이 큰 기능(멀티차트/볼륨프로파일)은 단계적으로 최적화

## 다음 우선순위

1. 드로잉 편집 고도화(핸들 드래그로 점 이동/라인 스타일 편집)
2. 성과 리포트 고도화(지표별 기여/기간별 분해)
3. 테마 및 워크스페이스 프리셋 UX 정리
4. 선물/옵션 데이터 지원 범위 재정의
