# TradingView vs Quanting — 기능 갭 분석

> TradingView.com 기준 기능 비교 및 Quanting에서 추가 구현 가능한 항목 정리
>
> 참고: [TradingView Features](https://www.tradingview.com/features/) · [Chart Types](https://www.tradingview.com/support/solutions/43000703407-chart-types-available-on-tradingview/) · [Drawing Tools](https://www.tradingview.com/support/solutions/43000703396-drawing-tools-available-on-tradingview/)

## 범례

| 기호 | 의미 |
|------|------|
| ✅ | 이미 구현됨 |

## 2. 드로잉 도구 (Drawing Tools)

현재 Quanting에는 드로잉 도구가 전혀 없음. lightweight-charts는 자체 드로잉 API가 없으므로 **캔버스 오버레이 또는 커스텀 플러그인**으로 구현해야 함.

### 2-1. 추세선 도구

| 기능 | 난이도 | 구현 방식 |
|------|:------:|----------|
| 추세선 (Trend Line) | 🟢 중 | SVG/Canvas 오버레이 + 마우스 드래그 |
| 수평선 (Horizontal Line) | 🟢 소 | `createPriceLine()` API 활용 |
| 수직선 (Vertical Line) | 🟢 소 | 커스텀 프리미티브 |
| 레이 (Ray) | 🟢 중 | 한 방향 무한 연장선 |
| 평행 채널 | 🟢 중 | 두 평행선 + 영역 채우기 |
| 크로스 라인 | 🟢 소 | 수평 + 수직선 조합 |
| 인포 라인 (거리/가격차 표시) | 🟢 중 | 두 점 간 가격/시간 차이 |

### 2-2. 피보나치 도구

| 기능 | 난이도 | 구현 방식 |
|------|:------:|----------|
| 피보나치 되돌림 (Retracement) | 🟢 중 | 두 점 선택 → 비율 수평선 자동 생성 |
| 피보나치 확장 (Extension) | 🟢 중 | 세 점 선택 → 확장 레벨 |
| 피보나치 팬 | 🟢 대 | 부채꼴 선 |
| 피보나치 채널 | 🟢 대 | 평행 채널 + 비율 |
| 피보나치 타임존 | 🟢 중 | 수직선 시리즈 |

### 2-3. 기타 도구

| 기능 | 난이도 | 설명 |
|------|:------:|------|
| 피치포크 (Pitchfork) | 🟢 대 | 3점 기반 Andrew's Pitchfork |
| 갠 팬/박스 | 🟢 대 | 각도 기반 선 |
| 엘리엇 파동 라벨 | 🟢 중 | 텍스트 마커 배치 |
| 직사각형 영역 | 🟢 중 | 가격/시간 범위 하이라이트 |
| 텍스트 주석 | 🟢 소 | 차트 위 텍스트 추가 |
| 가격 범위 측정 | 🟢 중 | 드래그로 가격/% 차이 표시 |
| XABCD 패턴 | 🟢 대 | 하모닉 패턴 드로잉 |
| 헤드앤숄더 | 🟢 대 | 패턴 시각화 |

### 드로잉 도구 구현 전략

```
Phase 1: 수평선/추세선/피보나치 되돌림 (가장 많이 사용)
Phase 2: 평행 채널/직사각형/텍스트 주석
Phase 3: 피치포크/갠/엘리엇/하모닉
```

필요 아키텍처:
- `src/stores/useDrawingStore.ts` — 드로잉 객체 상태 관리
- `src/components/DrawingCanvas.tsx` — SVG 오버레이 레이어
- `src/components/DrawingToolbar.tsx` — 좌측 도구 패널
- 드로잉 데이터 localStorage 영속화

## 3. 기술적 지표 (Technical Indicators)

### 현재 구현된 지표

| 지표 | 상태 | 비고 |
|------|:----:|------|
| 볼린저 밴드 (BB) | ✅ | 기간/승수 조절 가능 |
| SMA (최대 7개) | ✅ | 기간 자유 설정 |
| EMA (최대 7개) | ✅ | 기간 자유 설정 |
| RSI | ✅ | 기간 조절, 과매수/과매도 라인 |
| MACD | ✅ | 히스토그램 + 시그널 라인 |

| Parabolic SAR | 🟡 중 | ★★★★ | 추세 추적 점. 반전 신호 |

```

Tier 3 (보조): CCI, Williams %R, MFI, CMF, ROC, Aroon
Tier 4 (고급): Volume Profile
```

## 구현 로드맵 제안

### Phase 1: 핵심 분석 도구 강화 (높은 가성비)

```
1. 추가 지표: VWAP, ATR, Ichimoku, Supertrend, Parabolic SAR
2. 기본 드로잉: 수평선, 추세선, 피보나치 되돌림
3. 가격 알림 (시스템 노티피케이션)
4. 로그 스케일 / % 스케일 전환
5. 차트 비교 (심볼 오버레이)
```

### Phase 2: 프로 트레이딩 UX

```
1. 멀티 차트 레이아웃 (2~4분할)
2. 전략 성과 리포트 (승률, 수익률, MDD)
3. 볼륨 프로파일
4. 눈금자 도구 (측정)
5. 매수/매도 구간 시각화
```

### Phase 3: 고급 기능

```
1. 바 리플레이 (백테스팅)
2. WebSocket 실시간 데이터
3. 기술적 스크리너
4. 드로잉 도구 확장 (피치포크, 갠, 엘리엇)
5. 워크스페이스 저장/불러오기
```

### Phase 4: 데이터 확장
