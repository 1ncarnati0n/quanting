# Command Hub & Workspace Guide

## 목적
- 파워유저가 검색 중심으로 액션을 즉시 실행할 수 있도록 명령 허브를 제공한다.
- 분석 세션 상태(종목/지표/차트/드로잉)를 워크스페이스로 저장/복원한다.
- 실수 복구를 위한 안전 초기화 경로를 제공한다.

## 진입 경로
- 단축키: `Ctrl/Cmd + J`
- 구현 파일:
  - `src/components/CommandCenter.tsx`
  - `src/App.tsx` (전역 단축키 연결)
  - `src/utils/shortcuts.ts` (단축키 도움말 단일 소스)

## 핵심 기능
1. 명령형 액션 실행
- 심볼 검색 열기
- 설정/관심종목 패널 토글
- 차트 맞춤
- 리플레이 시작/종료
- 테마 전환
- 단축키 도움말 열기

2. 워크스페이스 저장/복원
- 현재 상태 저장: `quanting-workspaces` 로컬 스토리지에 저장 (최대 8개)
- 저장 상태: 종목/마켓/인터벌, 테마, 차트 타입/레이아웃, 설정 탭, 지표, 가격축, 비교 설정, 드로잉
- 복원 시 리플레이 종료 + 차트 맞춤 수행

3. 안전 초기화
- 확인 대화상자 후 실행
- 리플레이 종료
- 비교 차트 비활성화
- 가격축 기본(normal + autoScale) 복원
- 드로잉 전체 삭제
- 차트 맞춤

## 운영 규칙
- 워크스페이스는 사용자가 직접 삭제 가능
- 삭제/초기화는 항상 확인 대화상자를 요구
- 단축키/도움말/실제 동작은 `shortcuts.ts` 기준으로 유지

## QA 체크
- `Ctrl/Cmd + J`로 허브 열림 확인
- 검색어 입력 후 `Enter`로 첫 명령 실행 확인
- 워크스페이스 저장 후 종목/지표/드로잉 변경 → 복원 시 원상태 복귀 확인
- 안전 초기화 실행 시 확인 대화상자 노출 및 지정된 5개 동작 적용 확인
